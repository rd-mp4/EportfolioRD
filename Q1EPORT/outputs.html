Lesson 1: XHTML

XHTML (Extensible Hypertext Markup Language) is a stricter and more XML-compliant version of HTML that enforces well-formedness and proper nesting of tags.

Lesson 2: Lists

Lists in HTML can be created using ordered lists (ol) for numbered items, unordered lists (ul) for bullet points, and definition lists (dl) for terms and their definitions.

Lesson 3: Tables

Tables are used to display data in a structured format using the table element, which contains rows (tr), headers (th), and data cells (td).

Lesson 4: Merging Tables

Merging tables involves using the colspan and rowspan attributes to combine multiple columns or rows into a single cell.

Lesson 5: Hyperlinks

Hyperlinks are created using the anchor tag, allowing users to navigate to other web pages or resources by clicking on linked text or images.

Lesson 6: Forms

Forms in HTML allow users to submit data to a server and can include various input types such as text fields, checkboxes, radio buttons, and buttons.

Lesson 7: Forms - Text Box and Area, Radio Button

Text boxes and text areas enable users to enter text, while radio buttons allow users to select one option from a group.

Lesson 8: Forms - Check Boxes, File Extension, Submit and Clear Button

Check boxes enable multiple selections, file inputs allow users to upload files, and buttons can be created for form submission or to clear the form.
